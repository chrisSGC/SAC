Creer un compte

URL d'appel

/nouvelUtilisateur
POST
  • Requete
  • Reponse
Body

Media type: Application/json

Type: Object

Proprietes:

  • nomCompte: required (string)
  • motDePasse: required (string)

Exemple:

{
nomCompte : 'votre nom de compte',
motDePasse : 'votre mot de passe'
}

Reponse

Si echec:

{
nomCompte : 'votre nom de compte',
motDePasse : 'votre mot de passe'
}

Si succes:

{
nomCompte : 'votre nom de compte',
motDePasse : 'votre mot de passe'
}
